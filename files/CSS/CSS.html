    Gradient

    It's multi color background.

    Three types
  • 1. Linear
  • Syntax:
  • background: linear-gradient(direction,color1 %,color2 %,...,color n %)
  • In direction: [to left/ to right/ to top/ to bottom/ degrees like 30 deg]
    • Code: linear-gradient

  • 2. Radical
  • Syntax:
  • background: radical-gradient(shape,color1 %,color2 %,...,color n %)
  • In shape: [any shape] #default oval
    • Code: radical-gradient

  • 3. Conic
  • Syntax:
  • background: conic-gradient(color1 %,color2 %,...,color n %)
  • cone shape will come in middle of the screen
    • Code: Conic-gradient

    Position
  • The default position if web page: static

  • position: [absolute/ relative/ fixed/ sticky ]
  • absolute: starting of the container.
  • relative: adjust from previous tag.
  • sticky: fixed. Used for icons
    • CSS Selector
  • Selectors [style rules ] are used for to define styles for elements/ tags.
  • It means customizing existing styles for elements/ tags.
  • It can be defined only Internal css or External css
  • Syntax:    selector{ property, value};
    • Types of selectors:
  • 1. universal selector [*]
  • 2. id selector [#]
  • 3. class selector [.]
  • 4. tag selector
  • 5. child selector [space]
  • 6. direct child selector[>]
  • 7. attribute selector [ [] ]
  • 8. group selector [ , ]
  • 9. pseudo class selector [ : ]
  • 10. pseudo element selector [ :: ]
    • Flex
  • flex: row or column
  • flex-box & flex-items

    • Syntax:

  • display:flex; [#main attribute]

    • supportive attributes

  • flex-direction: [row #default/ row-reverse/ column, column-reverse]
  • gap: 10px; #in b/w flex items/li>
  • flex-wrap: [no-wrap/ wrap] #keep it one row
  • justify-content: [start/ end/ center/ space-between/ => space/(n-1)]
    space-around/ => space/n
    space-evenly =>space/(n+1)
    • Grid
  • N rows x 1 column #default
  • display:grid; [#main attribute]

    • Syntax:

  • grid-template-columns: [c1,c2] i.e [100px,200px/ 50%,50%/ auto,auto ]
  • grid-template-rows: [r1,r2] i.e [same as above ]

  • row-gap: 15px ; column-gap: 15px
    • or
  • grid-gap: 15px;
