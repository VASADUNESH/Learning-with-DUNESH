VARIABLES
  • Variable is a container is used to store the values in our programs.
  • Variable is a named memory location where we can store and manipulate [ modify ] the values in our programs.
  • Every variable is going to store in the primary memory i.e. RAM, which is temporary. Due to this the variable deleted automatically after the function / program execution.
  • In c compiler always the variables declared in first line only. In C++ we can declare anywhere.

Every variable is having 2 stages.

  1. Variable declaration / declared

    2. Eg: int a;

  2. Variable initialization / defined a=100;

When a variable is initialized then only memory allocated.

Syntax:

datatype variable[=value],

Eg: int id=1234, children=2; char name[ ]=”Ravi”, gender=’M’; float height = 5.8;
