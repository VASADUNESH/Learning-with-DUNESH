C-TOKENS

The smallest individual words we are using in developing a c program are called C – Tokens. They are of different types.

  • 1. Keywords: The system predefined / reserved words are called keywords. Every keyword is having certain meaning and as user we can’t change this meaning. C comes with 32 keywords.
    Eg: auto, break, continue, case, const, char, do, double, default, else, enum, extern, float, for, goto, int, long, while, switch, register, typedef, struct, union, static,….

  • 2. Constants:Fixed values are called constants. Constant value should be provided at the time of declaration only. i.e. further initializations not allowed. We can’t change a constant value during program execution

    • program
    Output:

    pi=3.140000

    program

    #include < stdio.h >

    int main()
    {
         const float pi=3.14;
         pi=30;
         printf("pi=%f",pi);

         return 0;
    }

    Output:

    Error:
    Cannot modify a const Object in function main


    Types of Constants
  • Numerical constants:
  • const int rollno = 1234;
  • const float pi = 3.14;

  • Character constants:
  • char name[10]=”Ravi”; string constant
  • char gender=’M’; char constant

    • program

    #include < stdio.h >

    int main()
    {
         const float pi=3.14;
         int rollno=1234;
         const char name[10]="Dunesh";
         const char gender='M';

         printf("Pi=%f\n",pi);
         printf("Rollnp=%d\n",rollno);
         printf("Name: %s\n",name);
         printf("Gender: %c\n",gender);

         return 0;
    }

    Output:

    pi=3.14000
    Rollno=1234
    name: Dunesh
    Gender: M


  • 3. Identifiers:Identifiers are nothing but names of variables, functions, files etc.
    • Identifier naming rules:
  • 1. Name should have to start with alphabet or underscore only [ _ ].
  • 2. Numbers allowed but not at first position.
  • 3. Spaces not allowed.
  • 4. No special char except underscore.
  • 5. Keywords not allowed.
  • 6. Identifiers are case sensitive i.e. lower and upper are different.
  • 7. Duplicate names are not allowed in same function or block{ }.
  • 8. Name may contain up to 32 characters and excess characters ignored by the compiler.
